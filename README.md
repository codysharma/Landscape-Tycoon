## An in-browser, clicker game
[Live version here]([https://pages.git.generalassemb.ly/codysharma-ga/hw-landscaper/](https://codysharma.github.io/Landscape-Tycoon/))

### Level 1:
Click to gain money, unlocking upgrades

### Level 2:
